插入排序法 (Insertion Sort)

一、運作原理

插入排序法 將資料分為「已排序」與「未排序」兩部分。最初，已排序部分只包含第一個元素。

演算法會逐一取回未排序部分的元素，並從右向左掃描已排序的部分，將該元素「插入」到正確的位置。在移動過程中，所有比該元素大的數都會向右移動一個位置。

二、運作過程圖解

範例陣列：[12, 11, 13, 5, 6]

輪次 待插入元素 排序過程與結果 說明
初始 - [12, 11, 13, 5, 6] 第一個元素視為已排序
Pass 1 11 [11, 12, 13, 5, 6] 11 比 12 小，移到 12 前面
Pass 2 13 [11, 12, 13, 5, 6] 13 比 12 大，位置不變
Pass 3 5 [5, 11, 12, 13, 6] 5 插入到數列最前面
Pass 4 6 [5, 6, 11, 12, 13] 6 插入到 5 和 11 之間

*註：粗體部分代表目前已排好序的範圍。

三、演算法偽代碼 (Pseudocode)

procedure insertionSort(A : list of items)
    for i := 1 to length(A) - 1 do
        key := A[i] // 選擇當前要插入的元素
        j := i - 1

        // 與左側已排序部分的元素比較
        while j >= 0 and A[j] > key do
            A[j + 1] := A[j] // 將較大的元素向右移
            j := j - 1
        end while

        A[j + 1] := key // 插入正確位置
    end for
end procedure
