氣泡排序法 (Bubble Sort)

一、運作原理

氣泡排序法 透過連續比較相鄰的兩個元素，並根據大小關係交換位置。每一輪完整的掃描後，該輪次中「最大」的元素會像氣泡一樣浮動到陣列的最末端。

二、運作過程圖解

範例陣列：[5, 3, 8, 4, 2]

輪次 排序結果 說明
Initial [5, 3, 8, 4, 2] 初始狀態
Pass 1 [3, 5, 4, 2, 8] 8 移至正確位置
Pass 2 [3, 4, 2, 5, 8] 5 移至正確位置
Pass 3 [3, 2, 4, 5, 8] 4 移至正確位置
Pass 4 [2, 3, 4, 5, 8] 排序完成

三、演算法偽代碼 (Pseudocode)

procedure bubbleSort(A : list of sortable items)
    n := length(A)
    repeat
        swapped := false
        for i := 1 to n-1 do
            // 如果左邊比右邊大
            if A[i-1] > A[i] then
                // 交換位置
                swap(A[i-1], A[i])
                swapped := true
            end if
        end for
        // 如果這一輪都沒有交換，代表已經排好了
    until not swapped
end procedure

上述偽代碼加入了 swapped 標記（Flag），這可以讓演算法在資料已經排序完成時提前結束，提高效率。

四、複雜度分析

最佳時間複雜度

O(n)

平均時間複雜度

O(n²)

最差時間複雜度

O(n²)

空間複雜度

O(1)
